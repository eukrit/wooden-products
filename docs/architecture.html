Security posture

  • Cloud Run: --no-allow-unauthenticated; only claude@ai-agents-go (gateway runtime SA) has roles/run.invoker.
  • Public surface: nothing. The previously-public salessheet.leka.studio domain has been re-purposed for a separate Next.js paddle-court site; the WPC fence platform is gateway-only.
  • Cost-quote endpoint sits behind require_approved (Firebase session + status=approved). Guests get 401/403; rejected/pending get 302 → /auth/pending.
  • Secrets: GSM (peak-api-token, xero-*, slack-bot-token, firebase-web-api-key, flask-session-key, gemini-api-key). No hardcoded credentials.
  • GCS: UBLA + publicAccessPrevention=enforced on products-wood-assets.
