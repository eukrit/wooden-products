Calculation Algorithm

1
Chargeable Weight / Revenue Ton
Freight is billed on the higher of actual weight or volumetric size — whichever earns more revenue for the carrier.
# Air freight
volumetric_kg = L_cm × W_cm × H_cm ÷ 6,000
chargeable_kg = max(actual_kg, volumetric_kg)

# Sea freight (LCL)
CBM         = L_cm × W_cm × H_cm ÷ 1,000,000
revenue_ton = max(CBM, actual_kg ÷ 1,000)
2
Freight Cost
Rate × chargeable unit. Sea rates are per CBM; air rates are per chargeable kg.
freight_cost = CBM × rate_per_CBM          # sea LCL
freight_cost = chargeable_kg × rate_per_kg  # air LCL
3
CIF Value
Cost + Insurance + Freight — the internationally recognised import valuation base. Used by Thai customs to calculate duty and VAT.
CIF = EXW_price + freight_cost + insurance_premium
4
Cargo Insurance (Chubb, via Logimark)
Rate confirmed from actual PO.1254 invoice (March 2026). Covers 110% of CIF value. Minimum premium USD 40.
insured_value = CIF × 110%
premium       = max(insured_value × 0.45%, USD_40_in_THB)
total_ins     = premium + stamp(4) + duplicate(1) + VAT(7%)
5
Thailand Import Duty + VAT
Duty is assessed on CIF value. VAT base includes duty (cascaded). HS code determines the duty rate.
import_duty = CIF × hs_duty_rate
VAT_base    = CIF + import_duty
VAT         = VAT_base × 7%
total_tax   = import_duty + VAT
6
Total Landed Cost
All costs from factory to Bangkok delivery address, in THB.
landed_cost = EXW + freight + insurance + import_duty + VAT
            + clearance_service_fee + last_mile_delivery
7
Minimum Selling Price (Areda / GO Corp rule)
Minimum 20% gross margin on landed cost. Installation priced separately.
min_sell = landed_cost ÷ (1 - margin)
         = landed_cost ÷ 0.80   # for 20% GM
         = landed_cost ÷ 0.75   # for 25% GM
         = landed_cost ÷ 0.70   # for 30% GM
