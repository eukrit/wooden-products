2. Core formula matrix

Planks per bay

planks_per_bay = floor((H_mm + G_mm) / (plank_face_mm + G_mm))
Effective plank face = 150 mm (nominal 161.5 mm − 11.5 mm tongue overlap). Verified against 3 Proforma Invoices: H=1.5 m → 10 planks · H=2.0 m → 13 · H=3.0 m → 20.

Post count by layout

LayoutFormula (for N bays)Example N=10
IN + 111
LN + 1 + corner_extra12 (+1 default)
UN + 1 + 2·corner_extra13
detachedN + runsN + R

Weight (kg)

net_kg = Σ (linear_m × kg_per_m) + Σ (count × kg_per_pc)
gross_kg = net_kg × (1 + packaging_overhead_pct)   # default 1.08

Shipping volume (m³)

raw_cbm = Σ (length × cross_section)
          + n_pedestals × 0.035 × 0.035 × 0.5
          + n_caps × 0.00008
          + n_lconn × 0.00005
shipping_cbm = raw_cbm × packing_factor             # default 1.25
